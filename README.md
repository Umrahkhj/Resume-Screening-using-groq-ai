# AI Resume Screening Assistant

An AI-powered Resume Screening application built using Python, Streamlit, and Groq LLM.

## Features
- Upload Resume (PDF)
- Paste Job Description
- AI-powered Resume Analysis
- Skill Matching
- Strengths & Weaknesses
- Hiring Recommendation

## Tech Stack
- Python
- Streamlit
- Groq API (LLM)
- PyPDF
- python-dotenv

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key
```
